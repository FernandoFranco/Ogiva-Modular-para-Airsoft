# Imagens dos Componentes

Esta pasta contém as imagens dos componentes eletrônicos utilizados na IHM de Configuração.

## Imagens Necessárias

- `teclado_4x4.jpg` - Foto do teclado matricial 4x4 ✅
- `lcd_1604a.jpg` - Foto do display LCD 1604A (16x4) ✅
- `pcf8574_io.jpg` - Foto do módulo PCF8574 I/O genérico (azul)
- `pcf8574_lcd.jpg` - Foto do módulo PCF8574 adaptador LCD
- `led_verde.jpg` - Foto do LED verde 5mm
- `buzzer.jpg` - Foto do buzzer ativo
- `conector_sata.jpg` - Foto do conector SATA fêmea 7 pinos
- `ds2431.jpg` - Foto do chip DS2431 (opcional)

## Formato Recomendado

- **Formato**: JPG ou PNG
- **Resolução**: 800x600 pixels (recomendado)
- **Tamanho**: Máximo 500KB por imagem
- **Fundo**: Branco ou neutro para melhor visualização

## Observações

- Use fotos reais dos componentes quando possível
- Mantenha boa iluminação e foco nas fotos
- Destaque o componente principal na imagem
- Os dois PCF8574 são diferentes: um é o módulo I/O genérico (azul) e outro é o adaptador específico para LCD
